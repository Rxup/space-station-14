## Подходы к переводу

### 1. Отсутствующие ключи (121 строка)
- **Подход**: Добавить ключи с английским текстом как временным решением
- **Инструмент**: Использовать `keyfinder.py` для автоматического добавления
- **Действие**: Перевести после добавления

### 2. Идентичные значения (6,012 строк)
- **Подход**: Перевести напрямую, так как текст уже есть в английской версии
- **Приоритет**: Высокий для критичных файлов, средний для остальных
- **Действие**: Перевести согласно приоритетам

### 3. Частично непереведённые (2,399 строк)
- **Подход**: Проверить каждую строку вручную
- **Причина**: Может содержать технические термины, которые должны остаться на английском
- **Действие**: Решить для каждой строки индивидуально

## Рекомендации по переводу

### Технические термины
- Оставить на английском: названия компонентов, технические параметры
- Перевести: описания, сообщения пользователю

### Имена собственные
- Оставить на английском: имена персонажей, названия мест (если не требуют локализации)
- Перевести: описания, пояснения

### Акценты и речь
- Рассмотреть сохранение оригинального стиля
- Перевести базовые слова, оставить специфичные замены
